# Брендинг vKOROBKU

Иконка изображает данные, сжимаемые внутрь коробки.

- `vkorobku-icon.svg` — векторный исходник;
- `vkorobku-icon.png` — превью 512×512;
- `src/vKOROBKU.App/Assets/vkorobku.ico` — Windows-иконка приложения.

Перегенерация ICO и PNG:

```powershell
python tools/generate_icon.py
```
