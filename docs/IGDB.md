# Обложки Steam и IGDB

Для найденных Steam-игр vKOROBKU сначала автоматически загружает вертикальную обложку из Steam CDN по App ID. Авторизация для этого не требуется. IGDB используется как резервный источник и для игр, добавленных вручную.

## Авторизация IGDB

IGDB API использует Twitch OAuth. В open-source desktop-приложение нельзя безопасно встроить общий Client Secret: любой пользователь сможет извлечь его из бинарного файла. Поэтому vKOROBKU использует собственные ключи пользователя.

1. Войти в [Twitch Developer Console](https://dev.twitch.tv/console/apps).
2. Зарегистрировать приложение.
3. Скопировать Client ID и создать Client Secret.
4. Нажать `IGDB` в vKOROBKU и сохранить значения.

Client Secret защищается Windows DPAPI в контексте текущего пользователя. В репозиторий, логи и кэш изображений секрет не попадает.

## Поиск

Игра ищется по названию через `POST /v4/games`. Из ответа используется `cover.image_id`, затем загружается изображение размера `cover_big_2x`. Запросы выполняются последовательно с задержкой, чтобы не превышать ограничения IGDB.

Поиск по названию иногда может выбрать обложку другого издания. В будущем будет добавлена ручная коррекция и сопоставление Steam App ID через external games.

## Кэш

Каталог: `%LOCALAPPDATA%\vKOROBKU\Covers`.

- изображения сохраняются по SHA-256 от Steam App ID и нормализованного названия;
- найденная в Steam или IGDB обложка используется локально без повторного запроса;
- отрицательный результат кэшируется на 7 дней;
- временный файл атомарно переименовывается только после полной загрузки;
- кнопка сохранения настроек IGDB запускает принудительное обновление.

Данные и изображения предоставляются IGDB.
