# vKOROBKU v0.1.2 — предварительный выпуск

Обновление для Windows 10/11 x64 с упором на простоту использования и поддержку игр, добавленных вручную.

## Главное

- добавлена автоматическая оптимизация одной кнопкой;
- программа сама анализирует игру и выбирает подходящий XPRESS/LZX-режим;
- добавлен необязательный экспертный режим с ручным выбором;
- технический журнал операций и обнаружение прерванной обработки;
- Steam Build ID и определение устаревшего анализа;
- распознавание частично сжатых игр после обновления.

## Игры, добавленные вручную

- ручные игры сохраняются между запусками;
- определение по GOG, `steam_appid.txt`, EXE, папке и каталогу Steam;
- исправлен выбор основной игры вместо DLC и бонусов;
- добавлена кнопка «Уточнить игру» для неоднозначных папок;
- улучшена загрузка обложек из нового Steam CDN;
- добавлен резервный поиск изображения через Steam App Details и IGDB.

## Установка

Скачайте `vKOROBKU-v0.1.2-win-x64.zip`, полностью распакуйте его и запустите `vKOROBKU.exe`. Файл `vKOROBKU.Worker.exe` должен находиться рядом.

> Это предварительная версия. Рекомендуется начинать с игр, которые можно восстановить через проверку файлов Steam.
