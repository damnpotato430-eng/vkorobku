# vKOROBKU v0.1.0 — предварительный выпуск

Первый публичный тестовый выпуск для Windows 10/11 x64.

## Возможности

- поиск библиотек Steam и ручное добавление игры;
- автоматические обложки Steam и резервный поиск IGDB;
- сохранение анализа и статуса сжатия;
- анализ выборки до 512 МБ, 1 ГБ или 2 ГБ;
- оценка экономии и скорости чтения;
- прозрачное XPRESS4K/8K/16K и LZX-сжатие;
- отдельный скрытый UAC-worker;
- прогресс, отмена и полная распаковка;
- определение ранее сжатых WOF/NTFS-игр.

## Установка

Скачайте ZIP для `win-x64`, полностью распакуйте его и запустите `vKOROBKU.exe`. Файл `vKOROBKU.Worker.exe` должен находиться рядом.

> Это предварительная версия. Рекомендуется начинать с игр, которые можно восстановить через проверку файлов Steam.
