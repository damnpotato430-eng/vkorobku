# vKOROBKU v0.1.3 — предварительный выпуск

Обновление надёжности для Windows 10/11 x64: исправлена распаковка, операции продолжаются с места остановки, статус сжатия определяется точнее.

## Исправления

- распаковка гарантированно снимает XPRESS/LZX-сжатие: используется документированное поведение `compact.exe` — два прохода `/U /EXE` и `/U` (раньше распаковка опиралась на недокументированный эффект флага `/F`);
- игры с несколькими предсжатыми файлами больше не считаются полностью сжатыми: статус определяется долей сжатых байт (от 85 % — «сжата», от 5 % — «сжата частично»), поэтому анализ и оптимизация снова доступны;
- распаковка, завершившаяся с ошибками, помечает игру «Сжата частично», а не «Не сжата»;
- проверка результата не считает ошибкой мелкие и несжимаемые файлы, которые `compact.exe` намеренно оставляет без сжатия.

## Новое

- прерванные операции продолжаются с места остановки: уже обработанные файлы пропускаются автоматически и сразу засчитываются в прогресс;
- при запуске приложение предлагает продолжить прерванное сжатие;
- частично сжатую игру теперь можно полностью распаковать — кнопка появилась в панелях оптимизации;
- приложение работает в одном экземпляре: повторный запуск активирует уже открытое окно;
- раздел «Операции»: текущая операция, её остановка и история последних задач;
- вручную добавленную игру можно убрать из библиотеки (файлы на диске не изменяются).

## Улучшения

- анализ выполняется во временной области на том же диске, где установлена игра, и не блокирует интерфейс;
- бенчмарк чтения делает несколько проходов для устойчивого результата;
- обложки загружаются с ограниченной параллельностью;
- осиротевшие временные каталоги анализа удаляются автоматически;
- секрет авторизации worker больше не передаётся через командную строку;
- добавлены модульные тесты и CI-сборка на GitHub Actions.

## Установка

Скачайте `vKOROBKU-v0.1.3-win-x64.zip`, полностью распакуйте его и запустите `vKOROBKU.exe`. Файл `vKOROBKU.Worker.exe` должен находиться рядом.

> Это предварительная версия. Рекомендуется начинать с игр, которые можно восстановить через проверку файлов Steam.
