# vKOROBKU v0.1.1 — предварительный выпуск

Обновление первой публичной версии для Windows 10/11 x64.

## Изменения

- скорость чтения теперь показывается в сравнении с несжатыми файлами;
- для каждого XPRESS/LZX-режима отображаются исходная скорость, скорость после сжатия и разница в процентах;
- старые результаты анализа предлагают повторное сканирование для получения сравнения;
- исправлено сохранение обложек после обновления библиотеки Steam;
- улучшено отображение состояния уже сжатых игр;
- добавлена фирменная иконка приложения;
- README дополнен ссылками на скачивание.

## Установка

Скачайте `vKOROBKU-v0.1.1-win-x64.zip`, полностью распакуйте его и запустите `vKOROBKU.exe`. Файл `vKOROBKU.Worker.exe` должен находиться рядом.

> Это предварительная версия. Рекомендуется начинать с игр, которые можно восстановить через проверку файлов Steam.
