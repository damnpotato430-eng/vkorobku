# vKOROBKU v0.1.4 — предварительный выпуск

Большое обновление для Windows 10/11 x64: дожатие обновившихся игр, наблюдение за сжатыми играми, пропуск несжимаемых файлов, защита DirectStorage-игр и заметно более читаемый интерфейс.

## Новое

- кнопка **«Дожать»** для частично сжатых игр: сжимает только оставшиеся файлы тем же алгоритмом — выбор методов не показывается, смешение алгоритмов исключено;
- **наблюдение за сжатыми играми**: при запуске (и по кнопке «Проверить сейчас») приложение проверяет, не деградировала ли экономия после обновлений, и показывает сводку «обновились и требуют дожатия — можно вернуть ~X ГБ»; пороги настраиваются;
- **пропуск заведомо несжимаемых файлов** (медиа, архивы — 41 тип): сжатие быстрее, прогноз анализа точнее, докачанные ролики не выглядят «деградацией»; свои расширения добавляются в настройках; `.pak` сознательно не пропускается;
- **детект DirectStorage**: бейдж на карточке, автоматическая оптимизация блокируется с пояснением (NTFS-сжатие ломает быстрый путь чтения), в экспертном режиме — после явного подтверждения;
- окно **«Настройки»** и технический журнал приложения в папке logs;
- **балансный выбор алгоритма**: программа больше не выбирает всегда XPRESS16K — алгоритмы с просадкой чтения хуже −15 % отбрасываются, а при разнице экономии до 400 МБ предпочитается более быстрый.

## Исправления

- накопленная экономия больше не обнуляется после повторного сжатия или дожатия;
- результаты анализа не дублируются и не попадают на чужую карточку при переключении игр во время анализа;
- размер на карточке обновляется фактическим размером папки (манифест Steam может учитывать не все приложения в общей папке);
- единичные несжимаемые файлы больше не удерживают игру в статусе «сжата частично» — итог определяется долей сжатых байт;
- понятное сообщение, если права администратора подтверждены другой учётной записью Windows;
- одиночная ошибка интерфейса больше не закрывает приложение (отчёт сохраняется в logs, работа продолжается).

## Интерфейс

- крупные кнопки действий и увеличенные тексты панели оптимизации;
- цветные теги статуса на карточках: зелёный «Сжата», оранжевый «Сжата частично», нейтральный «Не сжата»;
- экономия места по каждому накопителю и суммарно — в блоке накопителей;
- во время операции над прогрессом видно, какая игра обрабатывается, а выбранный режим показан в закреплённой карточке;
- контекстное меню карточки: «Перейти в каталог», «Проверить состояние сжатия», «Убрать из библиотеки»;
- переключение карточек стало мгновенным: свежий статус берётся из кэша без полного обхода файлов.

## Установка

Скачайте `vKOROBKU-v0.1.4-win-x64.zip`, полностью распакуйте его и запустите `vKOROBKU.exe`. Файл `vKOROBKU.Worker.exe` должен находиться рядом.

> Это предварительная версия. Рекомендуется начинать с игр, которые можно восстановить через проверку файлов Steam.
