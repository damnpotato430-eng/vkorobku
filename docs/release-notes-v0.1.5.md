# vKOROBKU v0.1.5 — предварительный выпуск

Удобство и обслуживание: библиотеку стало проще листать, добавлено окно «О программе» и уведомление о новых версиях.

## Новое

- **уведомление о новой версии**: при запуске приложение проверяет GitHub и, если вышел релиз новее, показывает кнопку со ссылкой на загрузку (без автоустановки);
- **поиск по библиотеке** — фильтр игр по названию по мере ввода;
- **сортировка библиотеки** — «По имени», «Сначала крупные» и «Сначала несжатые», чтобы быстро находить лучшие цели для сжатия;
- **окно «О программе»** — версия, ссылки на репозиторий, релизы и лицензию, ручная проверка обновлений и быстрый доступ к папке логов.

## Прочее

- сборка переведена в режим «предупреждения как ошибки» для чистоты кода;
- обновлены действия GitHub Actions в CI.

## Установка

Скачайте `vKOROBKU-v0.1.5-win-x64.zip`, полностью распакуйте его и запустите `vKOROBKU.exe`. Файл `vKOROBKU.Worker.exe` должен находиться рядом.

> Это предварительная версия. Рекомендуется начинать с игр, которые можно восстановить через проверку файлов Steam.
